# AIPiloty Desktop IDE — Architecture

> **Status**: Phase 1 scaffolded — Code OSS fork approach
> **Last updated**: 2026-07-22

---

## Product Decision: Owned Code OSS Fork

AIPiloty Desktop is a **forked build of Code OSS** (microsoft/vscode, MIT license) that
produces a **standalone desktop IDE I own and customize** — not an extension, not a
browser app in a window.

| Approach | Verdict |
|---|---|
| Electron + Next.js bridge | Rejected & removed — not a real IDE |
| VS Code extension only | For testing only — F5 in someone else's IDE |
| **Code OSS fork + bundled extension** | **CORRECT — own the shell** |

The AIPiloty extension (`desktop-ide/`) is built-in to the fork and provides AI features.

## Repository Layout

```
aipiloty/
├── code-oss-ide/                  # Code OSS fork scaffold (this is the product)
│   ├── bootstrap.sh               # Clone vscode, patch, copy extension, npm ci
│   ├── product.json               # AIPiloty branding (replaces vscode's product.json)
│   ├── .gitignore                 # Excludes vscode-fork/ (generated, not committed)
│   ├── scripts/
│   │   ├── install-deps.sh        # macOS prerequisites (Xcode, Node, Yarn, Python)
│   │   ├── run-dev.sh             # Launch IDE in dev mode (fast, no full build)
│   │   └── package-mac.sh        # Build standalone .app + .dmg (30-60 min)
│   ├── patches/                   # Optional git patches applied to vscode clone
│   └── extensions/                # Local extension overrides (generated by bootstrap)
│
├── desktop-ide/                   # AIPiloty VS Code extension (AI layer)
│   └── src/
│       ├── extension.ts           # Entry: commands, sidecar, status bar
│       ├── sidecar.ts             # Spawn/monitor FastAPI + Ollama
│       ├── keychain.ts            # VS Code SecretStorage
│       └── agent/
│           ├── chatProvider.ts    # Chat sidebar webview
│           ├── inlineEdit.ts      # Cmd+K command
│           ├── providerStatus.ts  # Status bar item
│           └── streaming.ts       # SSE client
│
├── backend/                       # FastAPI agent (unchanged)
└── frontend/                      # Next.js browser IDE (separate product)
```

## ProviderRouter

```
Claude (priority 10) → OpenAI (20) → Gemini (30) → Ollama (100, never suppressed)
Failover: on NEXT request (not mid-stream) for 429, quota, auth, timeout, overload
```

## How to Build (Apple Silicon Mac)

```bash
# ── Once: prerequisites ──────────────────────────────────────────
make fork-deps
# Installs: Xcode CLT, Homebrew, Node 20, Yarn 1.x, Python 3.11

# ── Once: clone + patch Code OSS (~10-20 min) ────────────────────
cd aipiloty/
make fork-install
# Clones microsoft/vscode, applies product.json branding,
# copies desktop-ide/ extension, runs npm ci

# ── Daily development (fast, no full build) ──────────────────────
make fork
# Starts AIPiloty IDE window + backend sidecar automatically

# ── Standalone .app build (30-60 min, only for releases) ─────────
make fork-package
# Output: code-oss-ide/AIPiloty-darwin-arm64/AIPiloty.app
```

## Customization Points

| What | Where |
|---|---|
| App name, ID, data folder | `code-oss-ide/product.json` |
| Extension gallery (Open VSX vs Microsoft) | `code-oss-ide/product.json` → `extensionsGallery` |
| App icon | `code-oss-ide/patches/` → icon patch |
| Default settings | `code-oss-ide/patches/settings.json.patch` |
| Additional built-in extensions | `bootstrap.sh` (copy to extensions/) |
| AI features (chat, inline edit, provider) | `desktop-ide/src/` |
| Agent tools, RAG, LLM routing | `backend/` |

---

## Security

| Surface | Control |
|---|---|
| API keys | `context.secrets` (OS keychain) — never in files |
| Backend | `127.0.0.1:8100` only |
| Telemetry | Disabled in `product.json` (`"enableTelemetry": false`) |
| Extension gallery | Open VSX (no Microsoft account) |

## Phase 4: Branded Release

```
1. Fork microsoft/vscode on GitHub → your own repo
2. Apply bootstrap.sh changes as commits (not patches)
3. npm run gulp vscode-darwin-arm64 → .app
4. Code-sign with Apple Developer certificate
5. create-dmg → AIPiloty.dmg
6. Distribute via GitHub Releases
```
